## Installation

```bash
npm install @importcsv/react zod
```

### Optional: Excel Support
```bash
npm install xlsx  # For .xls and .xlsx files
```

## Quick Start

```tsx
import { CSVImporter } from '@importcsv/react';
import { z } from 'zod';
import { useState } from 'react';

// Define your data schema
const schema = z.object({
  name: z.string().min(1),
  email: z.string().email(),
  company: z.string().optional()
});

function App() {
  const [isOpen, setIsOpen] = useState(false);

  return (
    <>
      <button onClick={() => setIsOpen(true)}>
        Import Customers
      </button>

      <CSVImporter
        modalIsOpen={isOpen}
        modalOnCloseTriggered={() => setIsOpen(false)}
        schema={schema}
        onComplete={(data) => {
          // data is typed: { name: string; email: string; company?: string }[]
          console.log(`Imported ${data.length} customers`);

          // Send to your API
          fetch('/api/customers/import', {
            method: 'POST',
            body: JSON.stringify(data)
          });
        }}
      />
    </>
  );
}
```

✅ **Automatic file parsing** - CSV, TSV, XLS, XLSX

## Framework Setup

### Next.js App Router

```tsx
'use client';
import { CSVImporter } from '@importcsv/react';
```

### Next.js Pages Router

```tsx
import dynamic from 'next/dynamic';

const CSVImporter = dynamic(
  () => import('@importcsv/react').then(mod => mod.CSVImporter),
  { ssr: false }
);
```

### Vanilla JavaScript

```html
<script src="https://unpkg.com/@importcsv/react@latest/build/index.js"></script>
<script src="https://unpkg.com/zod@latest/lib/index.umd.js"></script>
<script>
  const schema = z.object({
    name: z.string().min(1),
    email: z.string().email()
  });

  const importer = CSVImporter.createCSVImporter({
    domElement: document.getElementById('app'),
    schema: schema,
    onComplete: (data) => console.log(data)
  });

  importer.showModal();
</script>
```

## Performance Notes

### Large Files (1,000+ rows)
ImportCSV automatically optimizes for large datasets:
- **Virtual scrolling** renders only visible rows
- **Progressive validation** keeps UI responsive
- **Efficient memory usage** with optimized data structures

### Bundle Size
The entire library is ~100KB gzipped, including:
- Core importer functionality
- Virtual scrolling for large datasets
- All validation and transformation logic
- Auto-injected CSS styles

## Production Deployment

### Recommended Configuration

```tsx
import { z } from 'zod';

const schema = z.object({
  name: z.string().min(1),
  email: z.string().email(),
  company: z.string().optional()
});

<CSVImporter
  schema={schema}
  onComplete={handleComplete}
  primaryColor={brandColor}     // Match your brand
  darkMode={prefersDarkMode}    // Respect user preference
  modalCloseOnOutsideClick={true}  // Better UX
/>
```

### Error Handling

```tsx
const handleComplete = (data) => {
  try {
    // Process imported data
    processData(data);
    showSuccess(`Imported ${data.length} records`);
  } catch (error) {
    showError('Failed to process imported data');
    console.error(error);
  }
};
```
